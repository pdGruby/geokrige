# Installation

## Intro

GeoKrige requires Python 3.10+ and can be installed via PyPi with the following command:

```{bash}
pip install geokrige
```

## Dependencies

The package has some additional dependencies that need to be installed:

- `numpy` >= 1.24.3
- `geopandas` >= 0.14.1
- `scipy` >= 1.11.1
- `scikit-learn` >= 1.3.2
- `matplotlib` >= 3.7.2
- `pandas` >= 2.0.3
- `shapely` >= 2.0.2
- `rasterio` >= 1.3.9
